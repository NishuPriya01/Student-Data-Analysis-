# Student Performance Analysis

## Overview
This project analyzes student performance using Python.  
It explores how study hours, attendance, tutoring, and parental support affect GPA.

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn
- 
## Dataset Features
- Study Hours  
- Attendance  
- Tutoring  
- Parental Support  
- GPA
-  
## Analysis Performed
- Data loading and preprocessing  
- Conversion of categorical data into numeric values  
- GPA distribution analysis  
- Study hours vs GPA  
- Attendance vs GPA  
- Correlation heatmap  

## Visualizations
- GPA distribution (Histogram)  
- Study hours vs GPA (Scatter Plot)  
- Attendance vs GPA (Scatter Plot)  
- Correlation heatmap  

## Key Insights
- Students who study more tend to have higher GPA  
- Higher attendance improves performance  
- Tutoring has a positive effect on results  
- Parental support helps in better academic performance
- 
## 🚀 Conclusion
Student performance is influenced by study habits, attendance, and support systems.
